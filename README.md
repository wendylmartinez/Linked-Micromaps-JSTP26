# Linked-Micromaps-JSTP26
R Code to create graphics in the article "Using Linked Micromaps to Explore Complex Structures  in Official Statistics" by Randall Powers, Darcy Morris, John Eltinge, and Wendy Martinez. It is scheduled to appear in the Journal of Statistical Theory and Practice in 2026. A version of the article is also on arXiv. Just search for Randall Powers. 

The ability to view linked micromaps in color is important. Unfortunately, the print version of the article has to be in grayscale. The online version should have color figures. There are three folders in this repo containing the figures in three different color palettes. 

The authors also believe in reproducibility. So, the R code and data used to create the graphics are also included in separate folders.

Enjoy!
